# Conversor to bytes


## Following the examples:

```c#
using Conversor;

var result = ConversorBytes.Converter(float.MaxValue);

Console.Write(result);

256 PB
```

```c#
using Conversor;

var result = ConversorBytes.Converter(43242423);

Console.Write(result);

41,24 MB
```
## Install

```bash
  dotnet add package ConversorBytes
```
